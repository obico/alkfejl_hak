Részletes leírás

A robot teljes állapotleírása le egy adott időpillanatban.

A kommunikáció során ilyen objektumokat küldünk át a robot és a kliens között. A robottól érkezve az aktuális állapotot képviseli, míg az ellenkező irányba küldve parancsként a kívánt állapotot írja le.

A history ilyen objektumok sorozata.

Lásd még
Itt egy STATE viselkedési mintát véltem felfedezni. A state minta lehetővé teszi egy objektum számára, hogy megváloztassa a működését, ha a belső állapota változik.
A robotnak mindig van egy aktuális állapota, amit felhasználunk. Egyszerűen gombokkal tudjuk változtatni az állapotát. Ezt az aktuális állapotot, az állapot le tudja cserélni.
Láthatjuk a RobotProxy állapotokat.

Enumeráció-tagok dokumentációja

enum RobotState::Status
strong

A robot állapota.

Enumeráció-értékek
Default 

Alapállapot. A robot tartja az aktuális gyorsulást.

Reset 

Reset. Parancsként használjuk.

Stopping 

Stop parancs. Ez az állapot akkor is érvényes marad, ha megállt a robot.

Accelerate 

Gyorsítási parancs a robotnak. A gyorsítás mértékét a robot átveszi a RobotState::a tulajdonságból.

Error 

Hiba jelzésére szolgáló állapot. Akkor lép ide a robot, ha valamilyen probléma lép fel. Például túlfeszültség.

SelfTest 

Öntesztelés állapota
