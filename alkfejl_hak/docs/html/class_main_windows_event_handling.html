Részletes leírás

Ez az osztály tartalmazza a felhasználó felület (GUI) eseménykezelőit.

A függőségeit a konstruktoron kereszül kapja meg és kapcsolódik a GUI signaljaihoz.

A ConnectQmlSignals() metódust meg kell hívni a QML gyökér objektummal, hogy a megfelelő signalokat be tudja kötni.
