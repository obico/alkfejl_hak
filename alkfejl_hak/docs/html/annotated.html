Osztálylista
Az összes osztály, struktúra, unió és interfész listája rövid leírásokkal:
 CCommunicationStream alapú kommunikációs ősosztály
 CCommunicationSerialPortSoros port kommunikáció
 CCommunicationTcpSocketTCP socket alapú kommunikációra kialakított leszármazottja az általános Communication osztálynak
 CCommunicationTcpSocketClientQTcpSocket alapú kommunikáció kliens oldala
 CCommunicationTcpSocketServerTCP/IP szerver socket
 CMainWindowsEventHandlingEz az osztály tartalmazza a felhasználó felület (GUI) eseménykezelőit
 CRobotProxyA robot proxyja
 CRobotStateA robot teljes állapotleírása le egy adott időpillanatban
 CRobotStateHistoryVisszamenőleg tárolja a robot állapotait, valamint innen lehet lekérdezni a legutóbbi érvényes állapotot is
 CSimulatorA robot szimulátor
 CStvApplicationAlkalmazás osztály. A main() példányosítja és indítja el
